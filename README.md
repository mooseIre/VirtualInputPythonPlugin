# VirtualInputPythonPlugin

This repository contains the InputBridge driver source, Python caller wrapper, development memory files, and the packaged release zip.

## Release Package

- `InputBridge-Package.zip`
- The zip contains a single BAT entry point: `manage-driver.bat`.
- `uninstall-clean.ps1` is kept as a helper called by the BAT.

## Source

- `VirtualInput/` contains the driver source and Python wrapper.
- `VirtualInput/记忆库/` contains iteration notes.
